# Welcome

Ataraxy Labs builds some cool stuff that the world never saw before. 

1. [sem](https://github.com/ataraxy-labs/sem) - semantic version control CLI (diff, blame, graph, impact)
2. [weave](https://github.com/ataraxy-labs/weave) - entity-level semantic merge driver for Git
3. [inspect](https://github.com/ataraxy-labs/inspect) - entity-level code review triage + API service
4. [opensessions](https://github.com/ataraxy-labs/opensessions) - beautiful agent orchestration within tmux, a smart persistant sidebar
