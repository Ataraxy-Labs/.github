# Welcome

Ataraxy Labs builds some cool stuff that the world never saw before. 

1. [weave](https://github.com/ataraxy-labs/weave) - entity-level semantic merge driver for Git
2. [sem](https://github.com/ataraxy-labs/sem) - semantic version control CLI (diff, blame, graph, impact)
3. [inspect](https://github.com/ataraxy-labs/inspect) - entity-level code review triage + API service
4. [opensessions](https://github.com/ataraxy-labs/inspect) - just an internal tool our team uses to speed up iteration cycles.
